\# Volejbal Tatry — Project Brief



Volejbal Tatry is a modern web app for managing volleyball matches for a local group.



\## MVP goals

\- user registration and login

\- email/password and magic link authentication

\- match calendar and match list

\- player attendance response: going / not going

\- fixed teams: Tatry and Ostatní

\- one main match per event

\- result tracking only by sets, not points

\- team win/loss statistics

\- year-based filtering for statistics



\## Non-goals for MVP

\- payments

\- public portal

\- training 2v2 mode

\- point-by-point scoring

\- advanced attendance planning



\## Product style

\- mobile-first

\- modern sporty UI

\- dark blue, white, and turquoise accents



\## Technical direction

\- Next.js

\- TypeScript

\- Tailwind CSS

\- Supabase

\- clean architecture, easy to extend
